# House Marketplace

The app was deployed on Vercel platform.

Visit [Demo](https://deploy-marketplace-house.vercel.app)


Need to create a .env file in root directory

```js
REACT_APP_GEOCODE_API_KEY="YOUR_GEOCODE_API_KEY"
```

```js
npm install
npm start
```
